# Changelog

## v1.1 (2026-03-01)

### 버그 수정
- **컨텍스트 메뉴 미표시 문제 수정**: 브라우저 재시작 후 우클릭 메뉴("AI로 번역하기", "AI 문장 파헤치기")가 사라지는 버그 수정
  - 원인: `browser.runtime.onInstalled` 이벤트는 설치/업데이트 시에만 발화하므로, 브라우저 재시작 시 컨텍스트 메뉴가 재생성되지 않았음
  - 수정: 백그라운드 스크립트 로드 시마다 `removeAll()` 후 메뉴를 재생성하도록 변경

## v1.0 (2026-02-XX)

### 최초 출시 (AMO 제출)

#### 주요 기능
- 드래그한 텍스트 우클릭 → "AI로 번역하기"로 한국어 번역
- 단축키 `Ctrl+Shift+Y`로 번역
- 번역 결과 팝오버로 표시
- 우클릭 → "AI 문장 파헤치기"로 사이드바에서 상세 분석
  - 단일 문장: 문장 구조 + 직독직해 + 문법 노트
  - 여러 문장: 각 문장별 직독직해 + 개별 파헤치기 버튼
- Claude API 키 입력/저장 팝업
- 영어/일본어 → 한국어 지원
